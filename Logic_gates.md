# Logic Gates

## AND Gate
Output is 1 only if both inputs are 1.

## OR Gate
Output is 1 if at least one input is 1.

## NOT Gate
Flips the input bit.

## XOR Gate
Output is 1 if inputs are different.

## Why XOR Matters
XOR is commonly used in:
- Encryption
- Malware obfuscation
- Password masking
